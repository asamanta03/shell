Q1: What is Shell?
A: Shell is an intermediator between the operating system and terminal or user. It is the upper layer of the operating system that allows users to interact with the kernel and hardware.

Q2: Which shell is most commonly used in companies?
A: Bash (Bourne Again Shell) is the most popular and commonly used shell in real-time company environments.

Q3: How do you check which shell you are currently using?
A: Use the command: echo "$SHELL" or echo "$0" to display the current shell being used.

Q4: What is Shell Scripting?
A: Shell scripting is writing a series of Linux commands in a file with .sh extension in a specific order that gets executed sequentially. It is used for automation purposes.

Q5: How do you execute a shell script?
A: First give execution permission using "chmod +x filename.sh" or "chmod 755 filename.sh", then execute using "./filename.sh"

Q6: What is the difference between a variable and a function in shell scripting?
A: A variable stores values temporarily, while a function is permanent and contains a set of steps or commands that can be reused.

Answer: Use the command echo $SHELL or echo $0. This will display the current shell, typically showing /usr/bin/bash for bash shell.

Question 3: How do you execute a shell script?
Answer: First, give execute permission using chmod +x filename.sh or chmod 755 filename.sh, then run it using ./filename.sh

Question 4: How would you execute a PS command every 30 seconds, 3 times?
Answer: Use a for loop with the sleep command. For example:

for i in {1..3}
do
  ps
  sleep 30
done
Question 5: What is the real-time use case of the uptime command?
Answer: To validate if machines were restarted, especially useful for patching validation. For example, if you restart 100 machines over the weekend, you can check when each machine was last restarted using the uptime command.

Question 6: What is the difference between for loop and while loop?
Answer: For loop is used when you have a specific range or number of iterations (like 1 to 15), while a while loop continues as long as a condition is true.
